<h1 id="overview">
                jQuery ModalBox</h1>
<p>
    A powerful popup based on popular jQuery platform.
</p>
<h3 id="features">
    Features</h3>
<ol>
    <li>Responsive (Adjust width based on screen)</li>
    <li>Default options customizable</li>
    <li>Customizable UI via CSS3</li>
    <li>Two Beautiful Themes Avialable</li>
    <li>Photo Gallery(Mousewheel and Keyboard Navigation avialable)</li>
    <li>Auto Adjust or Set Custom Height/Width</li>
    <li>Show YouTube videos, Google Maps, SWF Files via Iframe</li>
    <li>Handle non existing URL</li>
    <li>Init, Show and Close callbacks</li>
    <li>Show Ajax and Inline content</li>
    <li>Set Modal Window</li>
    <li>Very Lightweight only 28kb</li>
    <li>Free Software License</li>
</ol>
